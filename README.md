# kilo
kilo is a free simple terminal text editor written in C.

clone the repository to a local directory and build:

```
git pull https://github.com/FMJ2000/kilo.git`
cd kilo
make
```

to start a new file:

```
./kilo
```

to open an existing file:

```
./kilo <filename>
```

shortcuts:

```
Ctrl+S - save
Ctrl+Q - quit
Ctrl+F - find
Ctrl+C - copy
Ctrl+V - paste
Ctrl+D - duplicate line
Ctrl+K - delete line
```
